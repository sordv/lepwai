Условные операторы

Условные операторы позволяют программе принимать решения на основе условий. Они выполняют определённые блоки кода только если указанные условия истинны. Это основа логики любого приложения.

Базовый оператор if

Самая простая форма условного оператора:

if (условие) {
// код, который выполнится, если условие истинно
действие;
}

Простой пример

let age = 18;
if (age >= 18) {
console.log("Вы совершеннолетний");
console.log("Можно голосовать");
}

Важно: В JavaScript условие в круглых скобках должно возвращать булево значение (true или false). Если возвращается не булево значение, JavaScript преобразует его в булево (truthy/falsy).

Оператор if-else

Добавляем альтернативный вариант выполнения:

if (условие) {
// выполняется, если условие истинно
действие_1;
} else {
// выполняется, если условие ложно
действие_2;
}
