Переменные

Переменные — это основа программирования. Представьте себе коробку, на которой написано имя, а внутри лежит какое-то значение. В Python переменные работают именно так: они хранят данные, которые можно использовать в программе.

Что такое переменная?

Переменная — это именованная область памяти, которая хранит данные. Когда вы создаёте переменную, вы резервируете место в памяти компьютера для хранения значения. Часто переменную представдяют как некую коробку, например "message", которая хранит в себе какой-то предмет, например "Hello!".

Важно: В Python не нужно объявлять тип переменной заранее. Python сам определяет тип данных на основе присвоенного значения.

Как объявлять переменные

Простое присваивание

Для создания переменной просто придумайте имя и используйте оператор = (равно):

# Создаём переменные
age = 25
name = "Анна"
temperature = 36.6
is_student = True

Множественное присваивание

В Python можно создать несколько переменных одновременно:

# Несколько переменных в одной строке
x, y, z = 10, 20, 30
# Присвоение одного значения нескольким переменным
a = b = c = 100

Правила именования переменных

  • Имя должно начинаться с буквы или символа подчёркивания (_)
  • Имя не может начинаться с цифры
  • Имя может содержать только буквы, цифры и подчёркивания (A-z, 0-9, _)
  • Имена чувствительны к регистру (age, Age и AGE — разные переменные)
  • Используйте понятные имена: user_name лучше, чем un

Примеры правильных имён:

user_name = "Иван"
_private_data = "секретно"
total_price = 1500.75
is_valid = True

Примеры неправильных имён:

# 1st_place = "первое" # Ошибка: начинается с цифры
# user-name = "Мария" # Ошибка: содержит дефис
# class = "Python" # Ошибка: class - зарезервированное слово

Как применять переменные

Использование в вычислениях

# Создаём переменные
price = 500
quantity = 3
discount = 0.1 # 10% скидка
# Вычисляем общую стоимость
total = price * quantity
total_with_discount = total - (total * discount)
# Выводим результат
print("Общая стоимость:", total)
print("Со скидкой:", total_with_discount)

Работа со строками

first_name = "Иван"
last_name = "Петров"
age = 30
# Конкатенация (сложение) строк
full_name = first_name + " " + last_name
# Форматирование строк
greeting = f"Привет, меня зовут {full_name} и мне {age} лет"
print(greeting)

Изменение значений переменных

Значение переменной можно изменять в любой момент:

counter = 0
print("Начальное значение:", counter) # Вывод: 0
# Увеличиваем значение
counter = counter + 1
print("После увеличения:", counter) # Вывод: 1
# Или более короткая запись
counter += 3 # То же, что counter = counter + 3
print("После += 3:", counter) # Вывод: 4

Основные типы данных

  • int — целые числа: 10, -5, 0
  • float — числа с плавающей точкой: 3.14, -0.5, 2.0
  • str — строки (текст): "Привет", 'Python'
  • bool — логический тип: True, False
  • NoneType — специальный тип для отсутствия значения: None

Совет: Чтобы узнать тип переменной, используйте функцию type(). Например: type(age) вернёт <class 'int'>
