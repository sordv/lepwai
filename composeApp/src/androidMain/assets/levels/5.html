Операторы, вычисления и операции над строками

Операторы — это специальные символы или ключевые слова, которые выполняют операции над переменными и значениями. В Python есть операторы для работы с числами, строками, сравнения и логические операторы.

Арифметические операторы

Используются для выполнения математических операций:

a = 15
b = 4
сумма = a + b # Сложение: 19
разность = a - b # Вычитание: 11
произведение = a * b # Умножение: 60
частное = a / b # Деление: 3.75
целочисленное = a // b # Целочисленное деление: 3
остаток = a % b # Остаток от деления: 3
степень = a ** b # Возведение в степень: 50625

Сокращённые операторы присваивания

x = 10
x += 5 # x = x + 5, теперь x = 15
x -= 3 # x = x - 3, теперь x = 12
x *= 2 # x = x * 2, теперь x = 24
x /= 4 # x = x / 4, теперь x = 6.0
x **= 2 # x = x ** 2, теперь x = 36.0

Операторы сравнения

Используются для сравнения двух значений. Возвращают True (истина) или False (ложь).

x = 10
y = 5
равно = x == y # False (10 равно 5?)
не_равно = x != y # True (10 не равно 5?)
больше = x > y # True (10 больше 5?)
меньше = x < y # False (10 меньше 5?)
больше_или_равно = x >= y # True (10 больше или равно 5?)
меньше_или_равно = x <= y # False (10 меньше или равно 5?)

Логические операторы

Используются для комбинирования условий:

age = 25
has_license = True
# and (И) - истинно, если оба условия истинны
can_drive = (age >= 18) and has_license # True
# or (ИЛИ) - истинно, если хотя бы одно условие истинно
has_discount = (age < 18) or (age >= 65) # False
# not (НЕ) - инвертирует значение
cannot_drive = not can_drive # False

Пример: Проверка входа в клуб

age = 20
has_id = True
is_vip = False
# Можно войти, если возраст >= 18 И есть ID ИЛИ если VIP
can_enter = (age >= 18 and has_id) or is_vip
print("Можно войти?", can_enter) # True

Операции над строками

Конкатенация (сложение строк)

greeting = "Привет" + " " + "мир" + "!"
print(greeting) # "Привет мир!"

Умножение строк

line = "-" * 30
print(line) # "------------------------------"

Индексация и срезы

text = "Python"
первый_символ = text[0] # "P" (индексация с 0!)
последний_символ = text[-1] # "n"
срез = text[2:5] # "tho" (символы с 2 по 4)
первые_три = text[:3] # "Pyt" (с начала до 3)
последние_три = text[-3:] # "hon" (последние 3 символа)

Методы строк

text = " Привет, Мир! "
без_пробелов = text.strip() # "Привет, Мир!"
верхний_регистр = text.upper() # " ПРИВЕТ, МИР! "
нижний_регистр = text.lower() # " привет, мир! "
замена = text.replace("Мир", "Python") # " Привет, Python! "
разделение = text.split(",") # [' Привет', ' Мир! ']
длина = len(text) # 15 (количество символов)

Форматирование строк

name = "Анна"
age = 25
height = 1.65
# f-строки (рекомендуемый способ)
message = f"Меня зовут {name}, мне {age} лет, мой рост {height:.2f} м"
print(message) # "Меня зовут Анна, мне 25 лет, мой рост 1.65 м"
# Метод format()
message2 = "Меня зовут {}, мне {} лет".format(name, age)
print(message2) # "Меня зовут Анна, мне 25 лет"

Приоритет операторов

Операторы выполняются в определённом порядке (от высшего к низшему):

  1. ** — возведение в степень
  2. *, /, //, % — умножение, деление, целочисленное деление, остаток
  3. +, - — сложение, вычитание
  4. ==, !=, >, <, >=, <= — сравнение
  5. not — логическое НЕ
  6. and — логическое И
  7. or — логическое ИЛИ
result = 5 + 3 * 2 ** 2 # Сначала 2**2=4, потом 3*4=12, потом 5+12=17
print(result) # 17
# Используйте скобки для изменения порядка
result2 = (5 + 3) * 2 ** 2 # Сначала 5+3=8, потом 2**2=4, потом 8*4=32
print(result2) # 32

Совет: Всегда используйте скобки для сложных выражений, даже если они не требуются. Это сделает ваш код понятнее и предотвратит ошибки.
