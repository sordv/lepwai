Переменные

Переменные в JavaScript — это контейнеры для хранения данных. Они позволяют сохранять информацию в памяти компьютера и использовать её в разных частях программы. Часто переменную представдяют как некую коробку, например "message", которая хранит в себе какой-то предмет, например "Hello!".

Способы объявления переменных

1. var (устаревший способ)

Использовался в старых версиях JavaScript. Имеет функциональную область видимости.

var age = 25;
var name = "Анна";
var isStudent = true;

Важно: В современном JavaScript рекомендуется не использовать var из-за особенностей области видимости.

2. let (современный способ)

Имеет блочную область видимости. Значение можно изменять.

let counter = 0;
counter = 10; // Можно изменить
let price = 99.99;

3. const (для констант)

Имеет блочную область видимости. Значение нельзя изменить после присваивания.

const PI = 3.14159;
// PI = 3.14; // Ошибка! Нельзя изменить константу
const APP_NAME = "Моё приложение";

Рекомендации по выбору:

  • Используйте const по умолчанию
  • Используйте let, если значение будет меняться
  • Избегайте var в новом коде

Правила именования переменных

  • Имя может содержать буквы, цифры, символы $ и _
  • Первым символом не может быть цифра
  • Регистр имеет значение (age и Age — разные переменные)
  • Нельзя использовать зарезервированные слова (if, else, function и т.д.)
  • Рекомендуется использовать camelCase: userName, totalPrice
// Правильные имена
let userName = "Иван";
const MAX_USERS = 100;
let $price = 50;
let _privateData = "скрыто";
// Неправильные имена

// let let = "слово"; // Зарезервированное слово

Типы данных в JavaScript

Примитивные типы:

  • string — строки: "Привет"
  • number — числа: 42, 3.14, -10
  • boolean — логические: true, false
  • undefined — неопределённое значение
  • null — пустое значение
  • symbol — уникальные идентификаторы (ES6+)
  • bigint — большие целые числа (ES2020+)

Объектные типы:

  • object — объекты: {name: "Иван"}
  • array — массивы: [1, 2, 3]
  • function — функции

Как узнать тип переменной

let name = "Анна";
let age = 30;
let isActive = true;
console.log(typeof name); // "string"
console.log(typeof age); // "number"
console.log(typeof isActive); // "boolean"
console.log(typeof undefinedVar); // "undefined"

Практические примеры использования

Математические вычисления

const price = 500;
const quantity = 3;
const discount = 0.1; // 10%
let total = price * quantity;
let finalPrice = total - (total * discount);
console.log("Общая стоимость:", total);
console.log("Со скидкой:", finalPrice);

Работа со строками

const firstName = "Иван";
const lastName = "Петров";
let age = 30;
// Конкатенация строк
const fullName = firstName + " " + lastName;
// Шаблонные строки (template literals)
const greeting = `Привет, меня зовут ${fullName} и мне ${age} лет`;
console.log(greeting);

Изменение значений

let counter = 0;
console.log("Начальное значение:", counter); // 0
// Увеличение значения
counter = counter + 1;
console.log("После увеличения:", counter); // 1
// Сокращённая запись
counter += 3; // То же, что counter = counter + 3
console.log("После += 3:", counter); // 4
// Инкремент
counter++; // Увеличивает на 1
console.log("После ++:", counter); // 5

value = true; // boolean
// Область видимости let vs var
if (true) {
let blockVar = "видна только в блоке";
var functionVar = "видна везде в функции";
}
// console.log(blockVar); // Ошибка!

Совет: Всегда объявляйте переменные с помощью let или const перед использованием. Использование необъявленных переменных создаёт глобальные переменные, что может привести к ошибкам.
